多元线性回归

在上一文中,我们用一元线性回归模型引出了成本函数的概念,介绍了梯度下降算法. 本文我们将对线性回归展开更普遍地讨论, 讨论在有多个特征值时候的线性回归是什么样的.

一、 多元线性回归模型

在上文中我们讨论的房子总价问题, 在同一地区房子总价主要就受房子的大小影响。

但是在现实生活中，很多问题不只受一个因素影响。

比如我们要评估一个消费者的购买能力，消费能力受他的年收入、年龄、教育水平等多个维度影响，不能单从一个方面考虑。并且这些因素的占比各不相同，于是我们可以定义这样一个模型来进行预测:

F(x1,x2,x3,...,xn)=w1x1+w2x2+w3x3+...+wnxn+bF(x_1, x_2, x_3,...,x_n) = w_1x_1+w_2x_2+w_3x_3 +...+w_nx_n + b F(x1​,x2​,x3​,...,xn​)=w1​x1​+w2​x2​+w3​x3​+...+wn​xn​+b

我们就得到了多元线性回归的模型。

例如我们现在有这样一组数据:

向量w称为回归系数， b称为截项距。

二、矢量化

想一下，如果要用代码实现

F(x1,x2,x3,...,xn)=w1x1+w2x2+w3x3+...+wnxn+bF(x_1, x_2, x_3,...,x_n) = w_1x_1+w_2x_2+w_3x_3 +...+w_nx_n + b F(x1​,x2​,x3​,...,xn​)=w1​x1​+w2​x2​+w3​x3​+...+wn​xn​+b

应该如何实现？

最简单的就是手动一个个相乘然后相加:

省力一点，可以用循环

这也没问题。但是有一点不太好，就是这是个串行的逻辑。当我们的n=1000000的时候，耗时会特别长。

我们可以把w和x用向量表示:

w⃗={w1 w2 w3 ... wn}x⃗={x1 x2 x3 ... xn}\vec{w} = \{w_1\ w_2\ w_3\ ... \ w_n \} \\ \vec{x} = \{x1\ x_2\ x_3\ ... \ x_n \} w={w1​ w2​ w3​ ... wn​}x={x1 x2​ x3​ ... xn​}

通过矩阵的点积我们就可以得到

w⃗⋅x⃗=w1x1+w2x2+w3x3+...+wnxn\vec{w}\cdot\vec{x} = w_1x_1+w_2x_2+w_3x_3+...+w_nx_n w⋅x=w1​x1​+w2​x2​+w3​x3​+...+wn​xn​

我们的模型也可以表示为

F(x⃗)=w⃗⋅x⃗+bF(\vec{x}) = \vec{w}\cdot\vec{x} +b F(x)=w⋅x+b

这样写不光看着清爽，在代码里的执行效率也会高很多。

通过numpy这个库的dot方法，能计算两个矩阵或向量的点积。并且底层的实现会调用硬件进行并行的运算，所以在大量数据的时候，运行耗时会少得多。

四、特征缩放

我们假设现在有这样一个线性回归的例子。特征值一共有两个，即

F(x1,x2)=w1x1+w2x2+bF(x_1,x_2) = w_1x_1+w_2x_2 +b F(x1​,x2​)=w1​x1​+w2​x2​+b

此时的x_1取值范围在[30, 2000], 而x_2的取值范围在[0, 5]

现在我们假设有这样一组数据:

x1=2000,x2=5,y=500x_1=2000, x_2=5, y=500 x1​=2000,x2​=5,y=500

此时w1和w2应该是什么样的？

w1应该是一个很小的值，w2应该是一个比较大的值，比如

w1=0.1,w2=40,b=100w_1=0.1, w_2=40, b=100 w1​=0.1,w2​=40,b=100

这样能满足

0.1∗2000+40∗5+100=5000.1*2000 + 40*5 + 100 = 500 0.1∗2000+40∗5+100=500

但是这样有什么问题呢？因为x1的取值特别大，x2的取值特别小，就导致了w1通常需要一个比较小的值，w2是一个比较大的值。

这时候我们画成本函数出来的话，会是一个很瘪的椭圆

这时候的成本对w1非常敏感， w1一个很小的变化，都可能导致成本函数变化很大，这时候执行梯度下降的时候，很有可能要经过更多的步数才能到达最低点。

而如果w1和w2取值范围差不多，那么成本函数应该很圆。

所以我们希望特征值的取值范围都差不多，这样能够加快梯度下降算法的执行。

所以，接下来介绍几种将特征值进行缩放的方法。

1.最大值缩放

上面我们的x1取值范围[30, 2000], x2取值范围在[0,5]

我们可以分别让他们除以最大值

x1′=x1/2000x2′=x2/5x_1'=x_1/2000 \\ x_2'=x_2/5 x1′​=x1​/2000x2′​=x2​/5

这时候取值范围就变成了

x1′∈[0.015,1]x2′∈[0,1]x_1' \in [0.015 ,1]\\ x_2' \in [0, 1] x1′​∈[0.015,1]x2′​∈[0,1]

这样我们就把x1和x2的取值范围都控制在了[0,1]之间。

2.均值归一化

这个方法我们需要先算出x1, x2的平均值，然后执行

x1′=x1−μ1max1−min1x2′=x2−μ2max2−min2其中μ1,μ2为x1,x2的平均值x_1' = \frac{x_1 - \mu_1}{max_1-min_1} \\ x_2' = \frac{x_2 - \mu_2}{max_2-min_2} \\ 其中\mu_1,\mu_2为x_1, x_2的平均值 x1′​=max1​−min1​x1​−μ1​​x2′​=max2​−min2​x2​−μ2​​其中μ1​,μ2​为x1​,x2​的平均值

这样可以把x1,x2的取值范围控制在[-1,1]之间

3.Z-score normailization

这个方法需要先计算出x1,x2的平均值和标准差，然后执行

x1′=x1−μ1σ1x2′=x2−μ2σ2其中μ1,μ2为x1,x2的平均值,σ1,σ2是x1,x2的标准差。x_1' = \frac{x_1 - \mu_1}{\sigma_1} \\ x_2' = \frac{x_2 - \mu_2}{\sigma_2} \\ 其中\mu_1,\mu_2为x_1, x_2的平均值,\sigma_1,\sigma_2是x_1,x_2的标准差。 x1′​=σ1​x1​−μ1​​x2′​=σ2​x2​−μ2​​其中μ1​,μ2​为x1​,x2​的平均值,σ1​,σ2​是x1​,x2​的标准差。

可以看到，在一定次数的迭代之后，成本下降得已经很小了。这时候我们认为梯度下降已经收敛了。

梯度下降的迭代次数可以写死，比如100000次或1000000次，但是收敛之后的迭代对成本的下降效果不明显。

所以我们可以认为，当一次迭代后成本下降值不超过某个阈值，认为已经收敛可以停止迭代了。

不过这个阈值在不同的数据里选择不同， 所以我们也可以根据成本变化的函数图像来确定梯度下降何时收敛的。

六、学习率的选择

选择一个良好的学习率是对于梯度下降来说是很重要的。

  • 过大的学习率可能会导致成本不收敛。
  • 过小的学习率会导致梯度下降运行速度变慢。

那么如何选择一个合适的学习率呢？我们可以调试。

首先先选一个特别小的值作为学习率（比如0.001），观察成本变化图像是否下降。然后没慢慢增大学习率(比如 * 3)，继续观察成本变化图。一直到找到一个学习率使得成本变化图有增加也有减小， 类似这样：

就可以认为此时的学习率过大了。

那我们就可以基于此选择一个尽可能大的学习率，保证成本变化图像始终是下降的。

七、特征工程

假设目前我们有一组关于土地的长和宽，和该土地转让费数据。

这里我们有2个特征值，所以可能我们的线性回归模型是这样的:

F(x1,x2)=w1x1+w2x2+bF(x_1,x_2) = w_1x_1 + w_2x_2+b F(x1​,x2​)=w1​x1​+w2​x2​+b

但是，你可能会认为土地面积也是影响最终费用的一个因素

所以我们可以令

x3=x1x2x_3 = x_1x_2 x3​=x1​x2​

然后把模型改为

F(x1,x2,x3)=w1x1+w2x2+w3x3+bF(x_1, x_2, x_3) = w_1x_1 + w_2x_2+w_3x_3+b F(x1​,x2​,x3​)=w1​x1​+w2​x2​+w3​x3​+b

这时候这个模型的回归效果可能会更好。

这样的改变我们成为特征工程。我们可以通过转换、组合等方式来添加特征，使得模型变得更好

八、多项式回归

我们再介绍特征工程中的一种——多项式回归。

假设现在有这样一组关于房子尺寸和房子价格的数据

可以看到，此时我们如果用一条直线F(x) = wx + b来拟合，最终效果肯定不好

那我们可以尝试加上一个x^2作为特征值。

f(x)=w1x+w2x2+bf(x) = w_1x + w_2x^2+b f(x)=w1​x+w2​x2+b

但是这样也不行，因为在x很大的时候，预测值最终会下降

那我们再尝试加上x^3作为特征值

f(x)=w1x+w2x2+w3x3+bf(x) = w_1x+w_2x^2+w_3x^3+b f(x)=w1​x+w2​x2+w3​x3+b

此时的图像可能更合适。

我们也可以加上根号x

f(x)=w1x+w2x+bf(x) = w_1x + w_2\sqrt{x}+b f(x)=w1​x+w2​x​+b

图像也很合适

所以，选择一个合适的模型，会让我们的线性回归拟合效果更好。

**注意，在多项式回归中，你通常需要进行特征缩放。**因为如果x在[0,10]，那x²的范围就是[0, 100], x³的范围就在[0, 1000]， 通常需要特征缩放后再进行梯度下降。
