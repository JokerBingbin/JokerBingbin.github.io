逻辑回归

1. 分类

前面两篇文章介绍了监督性机器学习中的回归中的线性回归，接下来我们开始另一个类别: 分类。

回顾一下分类和回归的区别。

  • 回归是从连续的值中预测出一个值，这里有无线的可能。
  • 分类是从有限的值中预测出一个值。

预测房价是回归，预测一封邮件是否是垃圾邮件是分类;

预测衣服的价格是回归，预测一件衣服是衬衫还是T恤是分类。

分类的候选值有两个或两个以上，但是是有限的。

2.逻辑回归

当分类的候选只有两个值时， 我们称为二元分类。比如:

一封邮件是否是垃圾邮件 否/0/false 是/1/true
一笔交易是否是金融诈骗 否/0/false 是/1/true
一个视频是否涉及违规 否/0/false 是/1/true
一个肿瘤是否是恶行肿瘤 否/0/false 是/1/true

通常我们会用0和1，或者false和true来表示两个选项。这里的0和1含义时negative和positive，但是没有好坏之分。只是针对问题作出两个选项。

对于这一类问题，我们用线性回归显然就无法太好的拟合。所以我们引入逻辑回归。

2.1 sigmoid函数

我们先介绍一下逻辑回归里的重要函数:sigmoid函数。

他的定义是:

g(z)=11+e−zg(z) = \frac{1}{1+e^{-z}} g(z)=1+e−z1​

我们来看看这个函数的函数图像

可以看到，当z越大时，g(z)越接近1。当z越小时，g(z)越接近0.

而当z=0的时候，此时的g(z) = 0.5.

所以有

g(z)∈(0,1)g(0)=0.5g(z)\in(0, 1)\\ g(0)=0.5 g(z)∈(0,1)g(0)=0.5

2.2 逻辑回归模型

sigmoid函数的范围在(0,1)， 恰好我们的二元分类问题也只有0和1两种输出。

于是我们借助sigmoid函数来引入逻辑回归模型。

f(x⃗)=g(w⃗⋅x⃗+b)=11+e−(w⃗⋅x⃗+b)f(\vec{x}) = g(\vec{w}\cdot\vec{x} + b)= \frac{1}{1+e^{-(\vec{w}\cdot\vec{x} + b)}} f(x)=g(w⋅x+b)=1+e−(w⋅x+b)1​

这个模型的含义可以表示为

f(x⃗)=P(y=1∣x⃗;w⃗,b)f(\vec{x}) = P(y=1|\vec{x};\vec{w},b) f(x)=P(y=1∣x;w,b)

也就是y=1的概率。这个模型的作用是用来预测y=1的概率有多大的。

比如一封邮件，我们算出来f(x) = 0.7, 说明这封邮件有70%的概率是一封垃圾邮件，有30%的概率不是一封垃圾邮件。

2.3 决定边界

因为g(0) = 0.5， 所以我们可以认为在z>0时， 预测结果都是1;在z<0时，预测结果都是0.

也就是说

w⃗⋅x⃗+b>0时,y^=1w⃗⋅x⃗+b<0时,y^=0\vec{w}\cdot\vec{x} + b > 0时, \hat{y}=1 \\ \vec{w}\cdot\vec{x} + b < 0时, \hat{y} =0 w⋅x+b>0时,y^​=1w⋅x+b<0时,y^​=0

所以当

w⃗⋅x⃗+b=0时\vec{w}\cdot\vec{x}+b = 0时 w⋅x+b=0时

就是逻辑回归的边界线。

比如，对于这样一组数据

我们最终拟合出来的模型可能是

f(x1,x2)=11+e−(x1+x2+3)f(x_1,x_2) = \frac{1}{1+e^{-(x_1+x_2 + 3)}} f(x1​,x2​)=1+e−(x1​+x2​+3)1​

所以他的边界就是

x1+x2+3=0x_1+x_2+3=0 x1​+x2​+3=0

也就是这样一条直线。

如果数据在直线的左边，我们就可以认为y=0, 在直线右边就可以认为y=1.

x1+x2+3=0这条直线就是这里的边界。

2.4 成本函数

接下来我们来看看逻辑回归的成本函数。

首先如果用线性回归的成本函数，是不太理想的：

J(w⃗,b)=12m∑i=1m(f(x⃗)−y)2J(\vec{w}, b) = \frac{1}{2m}\sum_{i=1}^m{(f(\vec{x})-y)^2} J(w,b)=2m1​i=1∑m​(f(x)−y)2

因为我们这里f(x)的定义已经变了，所以最终的函数图像也会改变，可能是这样的

此时他已经不是一个凹图像了，他有多个局部最小值，这样我们就无法使用梯度下降来拟合了。

所以，我们引入一个Loss函数来表示预测值和真实值之间的损失。

Loss(fw⃗,b(xi⃗),yi)={−log(fw⃗,b(xi⃗))  if:yi=1−log(1−fw⃗,b(xi⃗))  if:yi=0Loss(f_{\vec{w}, b}(\vec{x^i}), y^i) = \begin{cases} -log(f_{\vec{w}, b}(\vec{x^i}))\ \ if:y^i=1\\ -log(1-f_{\vec{w}, b}(\vec{x^i}))\ \ if:y^i=0 \end{cases} Loss(fw,b​(xi),yi)={−log(fw,b​(xi))  if:yi=1−log(1−fw,b​(xi))  if:yi=0​

从图像里我们可以看到，当f(x)接近于1时， 此时的损失时非常小的。而当f(x)接近于0时，损失变得特别大。

当f(X)越接近0, 损失就越小；f(X)越接近于1， 损失就越大。

Loss函数我们可以简化为

L(fw⃗,b(xi⃗),yi)=−ylog(fw⃗,b(xi⃗))−(1−y)log(1−fw⃗,b(xi⃗))L(f_{\vec{w},b}(\vec{x^i}), y^i) = -ylog(f_{\vec{w}, b}(\vec{x^i}))-(1-y)log(1-f_{\vec{w}, b}(\vec{x^i})) L(fw,b​(xi),yi)=−ylog(fw,b​(xi))−(1−y)log(1−fw,b​(xi))

然后最终的成本函数可以表示为

J(w⃗,b)=1m∑i=1mL(fw⃗,b(xi⃗),yi)=−1m∑i=1m(ylog(fw⃗,b(xi⃗))+(1−y)log(1−fw⃗,b(xi⃗)))J(\vec{w}, b) = \frac{1}{m}\sum_{i=1}^mL(f_{\vec{w},b}(\vec{x^i}), y^i)\\ =-\frac{1}{m}\sum_{i=1}^m(ylog(f_{\vec{w}, b}(\vec{x^i})) + (1-y)log(1-f_{\vec{w}, b}(\vec{x^i})))\\ J(w,b)=m1​i=1∑m​L(fw,b​(xi),yi)=−m1​i=1∑m​(ylog(fw,b​(xi))+(1−y)log(1−fw,b​(xi)))

这就是逻辑回归的成本函数。

2.5 梯度下降

和线性回归的梯度下降相同，我们依然是通过求偏导来变化w和b。

wj=wj−α∂J∂wj    j=1...nb=b−α∂J∂bw_j = w_j -\alpha \frac{\partial J}{\partial w_j}\ \ \ \ j=1...n \\ b = b-\alpha \frac{\partial J}{\partial b} wj​=wj​−α∂wj​∂J​    j=1...nb=b−α∂b∂J​

经过推导后，可以得到

wj=wj−αm∑i=1m(fw⃗,b(x⃗i)−yi)xi    j=1...nb=b−αm∑i=1m(fw⃗,b(x⃗i)−yi)w_j = w_j - \frac{\alpha}{m}\sum_{i=1}^m(f_{\vec{w}, b}(\vec{x}^i) - y^i)x^i\ \ \ \ j=1...n \\ b = b-\frac{\alpha}{m}\sum_{i=1}^m(f_{\vec{w}, b}(\vec{x}^i) - y^i) wj​=wj​−mα​i=1∑m​(fw,b​(xi)−yi)xi    j=1...nb=b−mα​i=1∑m​(fw,b​(xi)−yi)

怎么样，看着是不是和线性回归的梯度下降很像。但是注意， 逻辑回归的f(x)定义已经变了，所以在实际计算中，要记得替换。
