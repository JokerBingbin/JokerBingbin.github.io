神经网络入门

1. 神经元和大脑

我们简单了解一下我们大脑是如何运作的。

我们大脑里有一个叫神经元的东西，几乎大脑里的信息都是神经元通过电信号交换的。

神经元的树突状体通过轴突线向其他神经元发送电信号脉冲。最终成千上万的神经元，组成了我们的大脑，让我们的大脑拥有了强大的能力。

让我们简化一下，我们用一个圆圈表示一个神经元，那么我们大脑里的活动就可以简化为各个神经元之间传递信息

当然实际的大脑活动肯定要比这个复杂得多。

2. 神经网络

当神经网络被发明时，最初的目的就是编写能够模仿人脑学习和思考方式的软件。虽然现在的神经网络和大脑的工作方式截然不同，但是还是借鉴了大脑中神经元传递信息的方式。让我们一起来看看。

假设现在我们是一个服装商家。我们希望能够通过机器学习，来预测一款服装是否能够成为畅销款。我们现在有一些数据:该服装的售价, 该服装的面料, 该服装的款式, 该服装同类型服装是否是畅销款，该服装的宣传力度…

那么需求就是我们现在有一些特征值，希望预测是否能成为畅销款。那么很自然想到前面学习的逻辑回归。

假设我们已经有了一个模型

f(x⃗)=sigmoid(w⃗⋅x⃗+b)f(\vec{x}) = sigmoid(\vec{w}\cdot\vec{x} + b) f(x)=sigmoid(w⋅x+b)

那么只需要将特征值输入，就能预测这款服装称为畅销款的概率

但实际上，要预测一款服装能否称为畅销款，根据特征工程我们更需要另外一些特征，比如性价比，知名度等信息。

而性价比和服装售价,面料有关，知名度和宣传力度，同类型服装是否是畅销款有关。

所以，我们可以在中间加一层预测，用售价、面料去预测性价比， 用宣传力度和同类型服装是否是畅销款去预测知名度， 然后再用性价比、知名度去预测该服装是否是会畅销。

这就是一个神经网络的例子。

2.1 神经网络层

现代的大部分神经网络的基本组成都是神经元层。这里的层是Layer。

在上面的例子中，我们在最后的预测前添加了一层用来预测数据。这就称为是神经元层，其中的每一个预测模型，称为神经元。

在上面的例子中，中间这一层是Layer1， 最后的输出层是Layer2。同时，因为我们只关注最后一层的输出，所以最后一层又叫输出层， 中间的神经元层我们不太在意，所以又称为隐藏层。

再来看这样一个例子

这是一个有着三层神经元层的神经网络，中间的Layer1和Layer2是隐藏层，Layer3是输出层。

我们可以看到所有的神经元都采用了sigmoid，也就是逻辑回归。这里采用的方法叫做激活函数。

Layer1层接受我们的输入，然后输出了a^[1]向量给Layer2层；同理Layer2层接受了来自Layer1层的输入，然后输出了a^[2]向量给Layer3层；最终Layer3层接受来自Layer2层的输入，给出最后的预测结果。

每一层输出的值称为激活向量。

如果我们把Layer2层放大来看，就是三个神经元各自进行预测

a⃗1[2]=sigmoid(w1⃗[2]⋅a⃗[1]+b1[2])a⃗2[2]=sigmoid(w2⃗[2]⋅a⃗[1]+b2[2])a⃗3[2]=sigmoid(w3⃗[2]⋅a⃗[1]+b3[2])\vec{a}^{[2]}_1 = sigmoid(\vec{w_1}^{[2]}\cdot\vec{a}^{[1]} + b_1^{[2]})\\ \vec{a}^{[2]}_2 = sigmoid(\vec{w_2}^{[2]}\cdot\vec{a}^{[1]} + b_2^{[2]})\\ \vec{a}^{[2]}_3 = sigmoid(\vec{w_3}^{[2]}\cdot\vec{a}^{[1]} + b_3^{[2]}) a1[2]​=sigmoid(w1​​[2]⋅a[1]+b1[2]​)a2[2]​=sigmoid(w2​​[2]⋅a[1]+b2[2]​)a3[2]​=sigmoid(w3​​[2]⋅a[1]+b3[2]​)

每个神经元有自己的参数和截项距。最终就能得到a^[2]向量。Layer m层的的计算就可以表示为

a⃗1[m]=sigmoid(w1⃗[m]⋅a⃗[m−1]+b1[m])a⃗2[m]=sigmoid(w2⃗[m]⋅a⃗[m−1]+b2[m])...a⃗n[m]=sigmoid(wn⃗[m]⋅a⃗[m−1]+bn[m])\vec{a}^{[m]}_1 = sigmoid(\vec{w_1}^{[m]}\cdot\vec{a}^{[m-1]} + b_1^{[m]})\\ \vec{a}^{[m]}_2 = sigmoid(\vec{w_2}^{[m]}\cdot\vec{a}^{[m-1]} + b_2^{[m]})\\ ...\\ \vec{a}^{[m]}_n = sigmoid(\vec{w_n}^{[m]}\cdot\vec{a}^{[m-1]} + b_n^{[m]})\\ a1[m]​=sigmoid(w1​​[m]⋅a[m−1]+b1[m]​)a2[m]​=sigmoid(w2​​[m]⋅a[m−1]+b2[m]​)...an[m]​=sigmoid(wn​​[m]⋅a[m−1]+bn[m]​)

我们的输入x向量就可以看作是a^[0].

通过神经元组成神经元层，我们就可以创建一个强大的神经网络。

2.2 进行预测 向前传播

从前面的例子可以看出，这个神经网络是单向的。按照Layer1->Layer2->Layer3的顺序向前传播，最终到输出层给出预测。

其中每个神经元的激活函数，我们都可以根据需求使用线性回归，或者使用逻辑回归。

3.1 构建神经元层

在构建之前，我们先来看看要构建一个神经网络层需要哪些参数？

假设我们现在所有的神经元采用sigmoid函数作为激活函数，那我们需要接受的参数分别是a_in，W和b。

其中W是一个二维矩阵，记录了每一个神经元的参数。

例如一层有3个神经元，每个神经元接受2个特征值，那么W就是一个2*3的矩阵

W=[123357]W = \begin{bmatrix} 1 && 2 && 3 \\ 3 && 5 && 7 \end{bmatrix} W=[13​​25​​37​]

注意， 每一列代表一个神经元的参数。

在这个例子里，第一个神经元就应该是

sigmoid(1∗ain[1]+3∗ain[2]+b[1])sigmoid(1*a_{in}[1] + 3*a_{in}[2] + b[1]) sigmoid(1∗ain​[1]+3∗ain​[2]+b[1])

所以每一个神经元的预测值我们都可以通过取W的对应列来计算。
